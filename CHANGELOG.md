## 0.5.2

### Added

- Bottom dock — pill, play button, pinned instances, and pagination island — scales up with the launcher window so it doesn't feel undersized on big monitors.
- Active page indicator dot expands and shows the page number while you scroll, click, or wheel through pagination, then collapses back when you stop.

### Changed

- Auto-update installer runs in passive mode now: a brief progress dialog appears, no clicks required, the launcher relaunches itself. Replaces the old basic-UI mode that walked users through the full install wizard on every update.

### Fixed

- The bottom dock no longer sits on top of card actions (Install buttons, mod tile metadata) at the end of long lists. Scrollable content reserves space for the dock so the last row always lands above it.
