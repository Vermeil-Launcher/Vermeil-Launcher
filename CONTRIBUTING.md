# Contributing

Vermeil is a solo project. The source code is public for transparency, not for collaboration.

## What is not accepted

- Pull requests
- Code contributions
- Commits from external users
- Unsolicited refactors or rewrites

PRs will be closed without review.

## What is welcome

- **Bug reports** — Open a GitHub Issue with steps to reproduce, expected vs actual behavior, and your OS + Vermeil version.
- **Feature suggestions** — Open a GitHub Issue describing what you'd like and why it would be useful.

## Building from source

If you want to build and run Vermeil locally for personal use, see [docs/DEVELOPMENT.md](docs/DEVELOPMENT.md).

Note: this is an AI-generated codebase. Builds may be unstable. See [DISCLAIMER.md](DISCLAIMER.md).
